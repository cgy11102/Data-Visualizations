# Kickstarter Campaign Success Predictors

## Project Overview
This project analyzes a massive dataset of Kickstarter campaigns to identify the key metrics that determine whether a crowdfunding attempt will succeed or fail. It is designed to act as a business intelligence tool for startups looking to optimize their launch strategies.

## Key Insights Explored
* **Funding Goal Thresholds:** At what dollar amount do success rates drop significantly?
* **Duration Optimization:** Does a longer campaign inherently lead to more funding, or is there a "sweet spot" for campaign duration?
* **Category Performance:** Which product categories (e.g., Tech, Tabletop Games, Design) have the highest inherent success rates?

## Technical Details
* **Software:** Tableau Desktop
* **File Type:** Unpackaged `.twb` (Source structure) / Packaged `.twbx` (Includes data)
* **To View:** Download the `.twbx` file and open it using [Tableau Reader](https://www.tableau.com/products/reader) or Tableau Desktop.
