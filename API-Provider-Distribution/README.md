# API Provider Distribution & Market Share

## Project Overview
This technical visualization maps the distribution, reliability, and market share of various API service providers across the B2B software ecosystem. It is designed to help technical product managers and engineering teams benchmark service providers and understand the backend service landscape.

## Key Insights Explored
* **Market Dominance:** Which API providers hold the largest market share across different tech sectors (e.g., payment processing, mapping, communication)?
* **Reliability Metrics:** Visualizing uptime, latency, and operational stability across different providers.
* **Ecosystem Mapping:** How fragmented or consolidated is the reliance on specific third-party API services within modern software architectures?

## Technical Details
* **Software:** Tableau Desktop
* **File Type:** Unpackaged `.twb` (Source structure) / Packaged `.twbx` (Includes data)
* **To View:** Download the `.twbx` file and open it using [Tableau Reader](https://www.tableau.com/products/reader) or Tableau Desktop.
