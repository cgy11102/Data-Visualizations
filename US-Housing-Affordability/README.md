# U.S. Housing Affordability Index

## Project Overview
This dashboard provides a macroeconomic analysis of the United States housing market. By tracking affordability metrics across different demographic groups and geographic regions, this project highlights critical market disparities, cost-of-living crises, and broader economic shifts.

## Key Insights Explored
* **Geographic Disparities:** Which specific regions or metropolitan areas are experiencing the fastest decoupling of median income from median home prices?
* **Macroeconomic Drivers:** How do shifting interest rates and historical inflation metrics correlate with homeownership accessibility?
* **Demographic Cohort Analysis:** Identifying which demographic groups are disproportionately impacted by current housing market conditions.

## Technical Details
* **Software:** Tableau Desktop
* **File Type:** Unpackaged `.twb` (Source structure) / Packaged `.twbx` (Includes data)
* **To View:** Download the `.twbx` file and open it using [Tableau Reader](https://www.tableau.com/products/reader) or Tableau Desktop.
